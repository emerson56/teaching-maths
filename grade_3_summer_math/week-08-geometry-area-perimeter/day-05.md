# Day 5: Same Area, Different Perimeter

## Goal

Distinguish area from perimeter and explore their relationship.

## Materials

Grid paper, square tiles, weekly assessment.

## Number Talk

Can two shapes have the same area but different perimeters?

## Main Lesson

1. Build rectangles with area 24.
2. Calculate each perimeter.
3. Build rectangles with perimeter 20.
4. Calculate each area.
5. Discuss why area and perimeter measure different attributes.

## Practice

- Complete the Week 8 assessment with FAST-style items.
- Find same-area/different-perimeter examples.
- Find same-perimeter/different-area examples.
- Correct any unit-label errors.

## Challenge

Among rectangles with area 24, which has the smallest perimeter? Explain why.

## Exit Check

Write one sentence using "square units" and one sentence using "units."
