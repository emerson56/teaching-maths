# Day 4: Quadrilaterals and Perimeter

## Goal

Classify quadrilaterals by attributes and find perimeter of polygons.

## Materials

Shape cards, ruler, geoboard or grid paper.

## Number Talk

Can a square also be a rectangle? Explain using attributes.

## Main Lesson

1. Sort quadrilaterals by sides, angles, and parallel sides.
2. Discuss parallelograms, rectangles, rhombi, squares, and trapezoids.
3. Define perimeter as distance around.
4. Measure and add side lengths.
5. Find an unknown side when perimeter is known.

## Practice

- Classify quadrilateral cards.
- Find perimeters.
- Draw a polygon with a given perimeter.
- Solve one missing-side problem.

## Challenge

Draw two different rectangles with perimeter 24. Which has the greater area?

## Exit Check

State one attribute every rectangle has and one attribute every square has.
