# Day 1: Distributive Property With Area Models

## Goal

Break apart a multiplication problem and justify the parts with an area model.

## Materials

Grid paper, square tiles, colored pencils.

## Number Talk

Solve `8 x 7` by breaking 7 into 5 and 2.

## Main Lesson

1. Draw an `8 x 7` rectangle.
2. Split it into `8 x 5` and `8 x 2`.
3. Record `8 x 7 = (8 x 5) + (8 x 2)`.
4. Split the same rectangle another way.
5. Discuss why both decompositions preserve total area.

## Practice

- Decompose `6 x 9` in two ways.
- Decompose `7 x 12`.
- Match area models to expressions.
- Find and correct an incorrect decomposition.

## Challenge

Solve `7 x 18` in two ways without standard long multiplication.

## Exit Check

Draw and label an area model proving `8 x 7 = 56`.
