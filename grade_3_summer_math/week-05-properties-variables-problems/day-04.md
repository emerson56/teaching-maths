# Day 4: Two-Step Problems With Four Operations

## Goal

Represent and solve two-step problems using all four operations.

## Materials

Problem-solving notebook, highlighters, bar-model paper.

## Number Talk

A box holds 8 rows of 6 pencils. Twelve pencils are used. What must be calculated first?

## Main Lesson

Use the full routine:

1. Retell.
2. Identify quantities and units.
3. Draw a bar model or array.
4. Identify the intermediate question.
5. Write equations with an unknown.
6. Solve and check.

## Practice

Solve:

- Multiply, then subtract.
- Add, then divide.
- Divide, then multiply.
- A problem with an unknown starting amount.

## Challenge

Write and solve a two-step problem that uses both division and addition and has answer 35.

## Exit Check

Circle the intermediate answer in one completed problem and state what it means.
