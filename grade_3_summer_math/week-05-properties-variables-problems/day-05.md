# Day 5: Cumulative Assessment

## Goal

Demonstrate understanding of place value, operations, properties, equations, and word problems.

## Materials

Cumulative assessment, grid paper, manipulatives available.

## Number Talk

Solve `12 x 7` using two different decompositions.

## Main Lesson

Briefly review:

- Estimate before computing.
- Draw when the structure is unclear.
- Use multiplication to check division.
- Substitute to check an unknown.

## Practice

- Complete a cumulative assessment for Weeks 1-5.
- Include computation, explanation, and FAST-style items.
- Correct errors and categorize them.
- Reteach one fragile skill immediately.

## Challenge

Create an equation puzzle with two different unknowns for an adult to solve.

## Exit Check

Write the strategy that has become most useful this summer and give an example.
