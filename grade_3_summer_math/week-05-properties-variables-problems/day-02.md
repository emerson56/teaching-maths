# Day 2: Multiplying by Multiples of 10

## Goal

Multiply one-digit numbers by multiples of 10 using place value and properties.

## Materials

Base-ten blocks, place-value disks, grid paper.

## Number Talk

If `6 x 4 = 24`, what is `6 x 40`? Why?

## Main Lesson

1. Interpret `3 x 20` as 3 groups of 2 tens.
2. Record 6 tens as 60.
3. Connect `3 x 20` to `(3 x 2) x 10`.
4. Avoid the weak rule "just add a zero" unless the student can explain the place value.

## Practice

- `4 x 30`
- `7 x 60`
- `9 x 80`
- `5 x 90`
- Missing factor: `__ x 40 = 240`

## Challenge

Find two one-digit by multiple-of-10 equations with a product of 360.

## Exit Check

Explain why `8 x 50 = 400`.
