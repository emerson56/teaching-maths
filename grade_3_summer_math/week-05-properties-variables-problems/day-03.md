# Day 3: Variables and Missing-Number Equations

## Goal

Use a letter or box to represent an unknown and solve equations by reasoning.

## Materials

Balance drawing, equation cards, counters.

## Number Talk

What number makes `n + 37 = 92` true?

## Main Lesson

1. Discuss the equal sign as "has the same value as."
2. Solve `n + 7 = 19`.
3. Solve `4 x n = 36`.
4. Solve `48 / n = 6`.
5. Check each value by substitution.

## Practice

- `n - 28 = 47`
- `6 x n = 54`
- `72 / n = 8`
- `n / 5 = 7`
- Write an equation for a word story.

## Challenge

Find all whole-number pairs that satisfy `a x b = 48`.

## Exit Check

Explain why the letter does not always mean the same number in different equations.
