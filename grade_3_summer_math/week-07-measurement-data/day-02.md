# Day 2: Length, Fractions of an Inch, and Line Plots

## Goal

Measure to halves and fourths of an inch and represent measurements on a line plot.

## Materials

Ruler, small objects, line plot template.

## Number Talk

Which is longer: `3 1/2` inches or `3 1/4` inches? Explain.

## Main Lesson

1. Review inch marks, half-inch marks, and quarter-inch marks.
2. Measure five objects to the nearest quarter inch.
3. Record measurements in a table.
4. Build a line plot.
5. Interpret the data with addition and comparison questions.

## Practice

- Measure 8-10 objects.
- Make a line plot.
- Answer questions using the line plot.
- Write one question someone else can answer from the plot.

## Challenge

Find three objects whose total length is as close as possible to 12 inches.

## Exit Check

Explain why `2/4` inch is the same as `1/2` inch.
