# Day 3: Mass, Volume, and Temperature

## Goal

Estimate and measure mass, liquid volume, and temperature using appropriate units.

## Materials

Measuring cups, kitchen scale, containers, thermometer if available.

## Number Talk

Would a full bathtub be measured in cups, pints, quarts, or gallons?

## Main Lesson

1. Discuss choosing sensible units.
2. Estimate and measure liquid volume.
3. Estimate and measure mass or weight.
4. Read temperatures in real contexts.
5. Solve one-step and two-step measurement problems.

## Practice

- Sort units by what they measure.
- Estimate before measuring.
- Solve addition/subtraction problems with like units.
- Write a measurement comparison.

## Challenge

Create a recipe for a pretend drink using cups and pints. Then double it.

## Exit Check

Name one unit for length, one for liquid volume, one for weight or mass, and one for temperature.
