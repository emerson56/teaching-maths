# Day 1: Time and Elapsed Time

## Goal

Tell time to the nearest minute and solve elapsed-time problems with number lines.

## Materials

Analog clock, open number lines, schedule cards.

## Number Talk

How many minutes are there from 2:37 to 3:00?

## Main Lesson

1. Read times on an analog clock.
2. Count minutes to the next hour.
3. Use a number line to solve `2:37 + 45 minutes`.
4. Break elapsed time into friendly jumps.
5. Check by reasoning about hours and minutes.

## Practice

- Find end times.
- Find start times.
- Find elapsed time across an hour.
- Create a short daily schedule.

## Challenge

Design a 45-minute afternoon schedule that starts at 2:37. Include three activities and show the end time.

## Exit Check

Solve: Practice starts at 4:48 and lasts 35 minutes. When does it end?
