# Day 5: Graphs and Data Displays

## Goal

Collect, display, and interpret data using scaled graphs, line plots, and simple circle graphs.

## Materials

Survey question, graph paper, colored pencils, weekly assessment.

## Number Talk

If each picture stands for 4 votes, how many votes do 6 pictures show?

## Main Lesson

1. Collect a small data set.
2. Make a scaled pictograph.
3. Make a scaled bar graph.
4. Make or read a simple circle graph showing category totals.
5. Compare data across categories.

## Practice

- Complete the Week 7 assessment.
- Interpret a scaled pictograph.
- Interpret a bar graph.
- Interpret a line plot.
- Answer a two-step data question.

## Challenge

Create a graph where the scale is 3. Write a question that requires multiplication and subtraction.

## Exit Check

Why must you read the key before answering questions about a scaled graph?
