# Day 4: Two-Step Addition and Subtraction Problems

## Goal

Solve two-step problems by identifying intermediate and final questions.

## Materials

Highlighters, problem-solving notebook.

## Number Talk

What must you know first in this problem: "Maya had 350 cards, bought 125, then gave away 98"?

## Main Lesson

Use this routine:

1. Read without calculating.
2. Retell the story.
3. Draw a model.
4. Identify the first hidden question.
5. Write and solve equations.
6. Check units and reasonableness.

## Practice

Solve four two-step problems:

- Join, then separate.
- Compare, then join.
- Find a missing part, then compare.
- One problem with irrelevant information.

## Challenge

Write a two-step problem whose answer is 426. Exchange it with an adult and check the solution.

## Exit Check

Complete: "The intermediate answer tells me..."
