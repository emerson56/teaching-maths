# Day 1: Addition With Regrouping

## Goal

Add within 1,000 using place-value models and an efficient written algorithm.

## Materials

Base-ten blocks, place-value chart, grid paper.

## Number Talk

Solve `268 + 135` by breaking apart one addend.

## Main Lesson

1. Build `468 + 275`.
2. Combine ones and trade 10 ones for 1 ten.
3. Combine tens and trade 10 tens for 1 hundred.
4. Record each trade in a written algorithm.
5. Check with estimation.

## Practice

- `356 + 287`
- `479 + 154`
- `608 + 295`
- `125 + 348 + 207`
- Write a story for one equation.

## Challenge

Place digits 1-6 into two three-digit addends to make a sum as close to 1,000 as possible.

## Exit Check

Explain what the small regrouped 1 means in one addition problem.
