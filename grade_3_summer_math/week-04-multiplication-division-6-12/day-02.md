# Day 2: Facts for 8 and 9

## Goal

Use doubling and 10-minus-1 strategies for 8s and 9s.

## Materials

Grid paper, array cards, mini whiteboard.

## Number Talk

How can `10 x 7` help with `9 x 7`?

## Main Lesson

1. Find 8s by doubling a 4s fact.
2. Find 9s as 10 groups minus 1 group.
3. Use arrays to justify both strategies.
4. Observe, but do not depend on, digit-sum patterns in 9s.

## Practice

- `8 x 6`, `8 x 7`, `8 x 9`, `8 x 12`.
- `9 x 4`, `9 x 7`, `9 x 8`, `9 x 12`.
- Write related division facts.

## Challenge

Find `9 x 37` mentally using the same strategy, then explain why it works.

## Exit Check

Show how `4 x 7` can help solve `8 x 7`.
