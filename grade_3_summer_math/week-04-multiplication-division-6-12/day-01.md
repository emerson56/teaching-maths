# Day 1: Facts for 6 and 7

## Goal

Derive 6s and 7s from easier facts.

## Materials

Array cards, grid paper, multiplication chart.

## Number Talk

Solve `6 x 8` using `5 x 8`.

## Main Lesson

1. Use `6 x n = 5 x n + n`.
2. Use `7 x n = 5 x n + 2 x n`.
3. Draw split arrays to prove both strategies.
4. Identify which 6 and 7 facts are already known from turn-around facts.

## Practice

- Derive `6 x 7`, `6 x 9`, `7 x 8`, and `7 x 12`.
- Write related division facts.
- Solve three missing-factor equations.

## Challenge

Show two different decompositions for `7 x 8` and decide which is mentally easier.

## Exit Check

Complete: `7 x 9 = (5 x 9) + (__ x 9)`.
