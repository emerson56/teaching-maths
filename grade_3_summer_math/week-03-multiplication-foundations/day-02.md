# Day 2: Arrays, Rows, and Columns

## Goal

Represent multiplication with arrays and connect turn-around facts.

## Materials

Square tiles, grid paper, colored pencils.

## Number Talk

How are `3 x 8` and `8 x 3` alike and different?

## Main Lesson

1. Build a 3-row, 8-column array.
2. Rotate it to show 8 rows of 3.
3. Discuss why the total stays 24.
4. Connect arrays to rectangles and future area work.
5. Name the commutative property informally as a turn-around fact.

## Practice

- Draw arrays for `4 x 6`, `2 x 9`, and `5 x 7`.
- Write two equations for each array.
- Find missing dimensions for arrays with 18 and 20 tiles.

## Challenge

How many different rectangular arrays can be made with 24 tiles? Rotations count as the same rectangle.

## Exit Check

Draw one array that proves `7 x 4 = 4 x 7`.
