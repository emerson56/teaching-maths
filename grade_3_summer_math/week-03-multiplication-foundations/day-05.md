# Day 5: Facts for 3 and 4

## Goal

Use known facts, doubling, and decomposition for 3s and 4s.

## Materials

Counters, arrays, weekly assessment.

## Number Talk

How can `2 x 7` help you solve `4 x 7`?

## Main Lesson

1. Build 3s as 2 groups plus 1 group.
2. Build 4s by doubling a double.
3. Compare skip-counting with derived-fact strategies.
4. Practice related division.

## Practice

- Complete the Week 3 assessment.
- Solve mixed facts through 5s and 10s.
- Explain two derived facts with drawings.
- Correct errors using arrays.

## Challenge

Find three different ways to solve `4 x 8` without counting by ones.

## Exit Check

Explain how to derive `3 x 9` from a fact you already know.
