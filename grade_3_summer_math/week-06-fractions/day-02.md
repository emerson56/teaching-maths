# Day 2: Fractions on Number Lines

## Goal

Locate fractions on a number line and interpret them as distances from zero.

## Materials

String or tape number line, fraction cards, rulers.

## Number Talk

Where would `1/2` be between 0 and 1?

## Main Lesson

1. Mark 0 and 1.
2. Partition the interval into equal fourths.
3. Label `1/4`, `2/4`, `3/4`, and `4/4`.
4. Emphasize that points, not spaces, represent numbers.
5. Repeat with thirds and sixths.

## Practice

- Place fractions on number lines.
- Identify missing labels.
- Match shape models to number-line locations.
- Explain why `4/4 = 1`.

## Challenge

Place `1/2`, `2/3`, and `3/4` on the same number line accurately.

## Exit Check

Draw a number line proving that `3/4` is greater than `2/4`.
