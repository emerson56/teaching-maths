# Day 3: Equivalent Fractions

## Goal

Generate and recognize equivalent fractions using models and number lines.

## Materials

Fraction strips, grid paper, number lines.

## Number Talk

How can two fractions with different numbers name the same amount?

## Main Lesson

1. Overlay `1/2` and `2/4` strips.
2. Mark both on a number line.
3. Show `2/3 = 4/6` with equal-sized wholes.
4. State that equivalent fractions occupy the same point.
5. Avoid teaching cross-multiplication.

## Practice

- Find equivalents for `1/2`, `1/3`, `2/3`, and `3/4`.
- Complete missing numerators or denominators using models.
- Sort equivalent and non-equivalent pairs.

## Challenge

Find as many fractions equivalent to `1/2` as possible with denominators no greater than 20. Describe the pattern.

## Exit Check

Prove `3/6 = 1/2` in two ways.
