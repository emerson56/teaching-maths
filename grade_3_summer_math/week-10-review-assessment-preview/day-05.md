# Day 5: Performance Task and Grade 4 Preview

## Goal

Apply Grade 3 mathematics in an integrated project and glimpse next year's ideas.

## Materials

Graph paper, play money, ruler, colored pencils, optional grocery advertisements.

## Performance Task: Plan a Mini Class Party

The student must:

- Set a budget.
- Choose quantities and calculate costs.
- Make a scaled graph of snack preferences.
- Use fractions to describe servings.
- Draw a table layout.
- Calculate area and perimeter.
- Write a clear explanation of the plan.

## Grade 4 Preview

Choose one:

- Multiply a two-digit number by one digit with an area model.
- Explore division with a larger dividend.
- Add fractions with the same denominator.
- Read and compare numbers beyond 10,000.

## Celebration

Review the beautiful-problems notebook and select:

- Favorite problem.
- Most improved skill.
- Problem the student is proudest of solving.

## Exit Check

Write one mathematical question the student wants to investigate next.
