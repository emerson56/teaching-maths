# Day 1: Bar-Model Word Problems

## Goal

Use bar models to expose the structure of challenging multi-step problems.

## Materials

Problem-solving notebook, ruler, colored pencils.

## Number Talk

What can a drawing reveal that an equation alone might hide?

## Main Lesson

1. Read a problem without calculating.
2. Identify the whole, parts, comparisons, and unknown.
3. Draw a labeled bar model.
4. Write equations from the model.
5. Solve and verify in context.

## Practice

- One part-whole problem.
- One comparison problem.
- One multiply-then-add problem.
- One problem with an unknown starting amount.

## Challenge

Three teams score 148 points altogether. Team A scores 17 more than Team B. Team C scores 52. Find one possible score pair for A and B. Is the answer unique?

## Exit Check

Draw a bar model for the hardest problem and explain each label.
