# Day 5: Student-Created Problem Set

## Goal

Create, solve, revise, and explain original mathematical problems.

## Materials

Problem-set template, colored pencils, optional store props.

## Number Talk

What makes a math problem interesting rather than merely long?

## Main Lesson

Create a small store with 8 items. Then write:

- One two-step money problem.
- One multiplication or division problem.
- One data question.
- One challenge problem with more than one possible answer.

## Practice

- Solve every created problem.
- Ask an adult to solve without hints.
- Revise unclear wording.
- Complete the Week 9 assessment.

## Challenge

Create a problem that appears difficult but has an elegant shortcut.

## Exit Check

Write the intended strategy beneath each problem.
