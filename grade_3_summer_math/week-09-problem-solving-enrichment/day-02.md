# Day 2: Logic Puzzles and Variables

## Goal

Use organized reasoning, elimination, and variables to solve logic problems.

## Materials

Logic grid, equation cards, notebook.

## Number Talk

If `a + b = 20` and `a` is 4 more than `b`, what could you try first?

## Main Lesson

1. Solve a small three-person logic puzzle.
2. Record clues in a table.
3. Cross out impossibilities.
4. Translate a clue into an equation.
5. Check that the final answer satisfies every clue.

## Practice

- One ordering puzzle.
- One missing-number puzzle.
- One balance puzzle.
- One cryptarithm with distinct digits.

## Challenge

Use different digits for A and B: `AB + BA = 99`. Find all solutions.

## Exit Check

State one fact you knew directly and one fact you deduced.
