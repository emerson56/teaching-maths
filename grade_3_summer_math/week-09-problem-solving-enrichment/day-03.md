# Day 3: Estimation and Fermi Problems

## Goal

Make reasonable assumptions, estimate quantities, and give useful bounds.

## Materials

Notebook, ruler, household objects.

## Number Talk

About how many steps does it take to cross a room? How could you estimate?

## Main Lesson

1. Ask: How many books could fit on one shelf?
2. List needed information.
3. Choose reasonable measurements.
4. Calculate an estimate.
5. State a lower and upper bound.
6. Test the estimate if practical.

## Practice

Estimate:

- Number of tiles on a floor.
- Number of cups in a container.
- Number of pages read in a month.
- Number of minutes spent eating breakfast in a year.

## Challenge

Estimate how many pencils would be needed to stretch across the length of the home. Defend the assumptions.

## Exit Check

Explain why a reasonable estimate can be valuable even when it is not exact.
