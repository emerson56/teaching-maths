# Grade 3 Summer Math Program

This is a rigorous 10-week summer program for a rising 3rd grader in Florida.

The program treats Florida B.E.S.T. standards as the floor, then layers in Singapore-style conceptual development, Beast Academy-style problem solving, and long-term preparation for mathematical competitions and later SAT success.

## How to Use This Folder

Start here:

1. Read [00-teaching-guide.md](00-teaching-guide.md).
2. Review [01-florida-alignment.md](01-florida-alignment.md).
3. Give the [02-placement-diagnostic.md](02-placement-diagnostic.md).
4. Teach one daily lesson file at a time.
5. Use [03-assessment-and-fluency.md](03-assessment-and-fluency.md) every Friday.
6. Use [04-competition-and-sat-foundation.md](04-competition-and-sat-foundation.md) twice per week.

## Printable Lesson Packets

Each lesson has a student packet and a scripted teacher guide in both DOCX and PDF formats.

- Week 1, Day 1 packets are in `week-01-place-value/materials/`.
- All other packets are in the week's `materials/day-XX/` folder.
- Student packets contain five kid-friendly pages with missions, worked examples, practice, challenge work, and reusable math tools.
- Teacher guides contain lesson theory, exact teacher language, material setup and use, misconceptions, differentiation, assessment guidance, and answer notes where appropriate.

## Time

Plan for 60-75 minutes per day, 5 days per week.

If you have only 8 weeks, combine Weeks 1-2 and Weeks 9-10 by trimming review and extension work.

## Non-Negotiables

- The child explains reasoning every day.
- The child sees multiple methods, not just one procedure.
- The child solves word problems every week.
- Timed fluency waits until strategies are secure.
- Competition math is playful, challenging, and low-pressure.
