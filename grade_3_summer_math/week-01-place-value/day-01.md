# Day 1: Place-Value Diagnostic and Language

## Goal

Read, build, and describe four-digit numbers using thousands, hundreds, tens, and ones.

## Materials

Base-ten blocks or place-value disks, four-column place-value chart, digit cards.

## Number Talk

How many different ways can you make 1,000 using hundreds, tens, and ones?

## Main Lesson

1. Give the placement diagnostic sections on number sense and mental calculation.
2. Build `3,245` with materials.
3. Record: 3 thousands, 2 hundreds, 4 tens, 5 ones.
4. Trade one thousand for 10 hundreds and discuss why the value stays equal.
5. Repeat with `4,070`, `6,308`, and a number chosen by the student.

## Practice

- Build and write `2,619`, `5,040`, and `7,503`.
- State the value of each digit.
- Write each number in words.
- Decompose `8,462` in two different ways.

## Challenge

Make a four-digit number that uses exactly 14 place-value pieces. Find more than one answer.

## Exit Check

Explain why the 5 in `5,204` is worth 100 times the 5 in `52`.
